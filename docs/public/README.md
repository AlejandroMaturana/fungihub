# Assets estáticos de Mush2 Docs

Este directorio contiene los archivos estáticos servidos directamente por VitePress
(favicon, imágenes, iconos, etc.).

Los archivos aquí colocados quedan disponibles en la raíz del sitio compilado.
Ejemplo: `public/favicon.ico` → accesible como `/favicon.ico`

Referencia: https://vitepress.dev/guide/asset-handling#the-public-directory
